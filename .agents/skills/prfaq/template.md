## Why Now?

*\[Business Case: Things to ideally cover even if it is a hypothesis that requires additional validation through the product discovery process\]*

- *How does this align to the strategic goals for the product line?*   
- *What is the current experience for the user and how will it change going forward?*   
- *How important is the user need and the user satisfaction with current alternatives?*   
- *Will this be an opportunity to disrupt current alternatives?*   
- *What is the value that it will bring to the business?*   
- *What is the cost to to the business to deliver this capability?*  
- *What percentage of the overall product addressable market will value this capability?*   
- *What would happen if we don’t invest in this capability?*
