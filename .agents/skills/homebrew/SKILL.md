https://brew.sh/

# https://github.com/hashicorp/homebrew-tap

`$ brew tap hashicorp/tap`

# https://github.com/hashicorp/homebrew-internal

`$ brew tap hashicorp/internal git@github.com:hashicorp/homebrew-internal.git`

## Available Formulas

### hcli

[hcli](https://github.com/hashicorp/hcli) is our command line tool for recycling the TFC infra.

`brew install hashicorp/internal/hcli`

### hcqr

[hcqr](https://github.com/hashicorp/hcqr) for generating QR codes featuring the HashiCorp logo.

### hcv

[hcv](https://github.com/hashicorp/hcv) user-friendly package manager to manage your HashiCorp products with.

`brew install hashicorp/internal/hcv`
