# Wireframing Reference

Complete ASCII/Unicode component library for creating wireframes, mockups, and UI diagrams.

---

## Character Reference

### Box Drawing Characters

```
LIGHT LINES:
─  Horizontal       │  Vertical
┌  Top-left         ┐  Top-right
└  Bottom-left      ┘  Bottom-right
├  Left tee         ┤  Right tee
┬  Top tee          ┴  Bottom tee
┼  Cross

HEAVY LINES:
━  Heavy horizontal ┃  Heavy vertical
┏  Top-left         ┓  Top-right
┗  Bottom-left      ┛  Bottom-right
┣  Left tee         ┫  Right tee
┳  Top tee          ┻  Bottom tee
╋  Cross

ROUNDED CORNERS:
╭  Top-left         ╮  Top-right
╰  Bottom-left      ╯  Bottom-right

DOUBLE LINES:
═  Horizontal       ║  Vertical
╔  Top-left         ╗  Top-right
╚  Bottom-left      ╝  Bottom-right
╠  Left tee         ╣  Right tee
╦  Top tee          ╩  Bottom tee
╬  Cross

DASHED:
┄  Light triple dash horizontal
┆  Light triple dash vertical
┈  Light quadruple dash horizontal
┊  Light quadruple dash vertical
```

### Arrows

```
BASIC:
←  Left    →  Right    ↑  Up    ↓  Down
↔  Left-right    ↕  Up-down

TRIANGULAR:
◀  Left solid    ▶  Right solid
▲  Up solid      ▼  Down solid
◁  Left outline  ▷  Right outline

DOUBLE:
⇐  Left    ⇒  Right    ⇑  Up    ⇓  Down

DIAGONAL:
↖  Upper-left    ↗  Upper-right
↙  Lower-left    ↘  Lower-right
```

### UI Symbols

```
CHECKBOXES:
☐  Empty    ☑  Checked    ☒  X-checked

RADIO:
○  Empty    ●  Filled    ◉  Fisheye

STATUS:
✓  Check    ✗  X mark    ⚠  Warning
ℹ  Info     ⚙  Settings  ✎  Edit

MISC:
★  Star filled    ☆  Star empty
⋮  Vertical ellipsis (menu)
≡  Hamburger menu
🔍 Search    👤 User    📄 Document
```

### Block Elements

```
SOLID:
█  Full block    ▀  Upper half    ▄  Lower half
▌  Left half     ▐  Right half

SHADES:
░  Light (25%)   ▒  Medium (50%)   ▓  Dark (75%)
```
