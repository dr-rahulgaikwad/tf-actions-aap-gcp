# hashi-designer Starter Kit

Learn hashi-designer by working through a real project.

## Setup

1. Load the skill: `use hashi-designer`
2. Navigate to `sample-project/`
3. Work through exercises in order

## What's Included

- **reference/prd.md** — Product requirements doc (IAC Remediation Tool)
- **reference/rfc.md** — Technical architecture
- **reference/uxr-raw-notes.md** — Raw interview notes (unprocessed)

## Exercises

| # | Exercise | What You'll Do |
|---|----------|----------------|
| 1 | User Research | Synthesize notes, generate insights, create principles |
| 2 | Digest PRD | JTBD, OOUX, user journeys, UXDR, propose UI |
| 3 | UI Scan | Heuristic evaluation, pattern checks |
| 4 | Documentation | Design principles, specs, handoff docs |

## Getting Started

```
use hashi-designer
```

Then open `exercises/01-user-research.md` and try the prompts.

## Tips

- Each exercise builds on the previous
- Save outputs to `sample-project/` as you go
- The raw notes are intentionally messy — that's the point
